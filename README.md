# ESP8266 WiFi Login Page using NodeMCU

## Description

This project demonstrates how to create a WiFi Access Point using the NodeMCU ESP8266 and host a login page. When users connect to the WiFi network, they can open a web browser and access the login page hosted directly on the ESP8266.

This project helps in understanding:

* ESP8266 Web Server
* WiFi Access Point mode
* Embedded Web Development
* IoT Networking Basics


## Features

* Creates its own WiFi network
* Hosts a login webpage
* Accepts user input
* Displays response page
* Runs completely on NodeMCU



## Hardware Required

* NodeMCU ESP8266
* USB Cable
* Computer or Mobile



## Software Required

* Arduino IDE
* ESP8266 Board Package
* CH340 or CP2102 Driver



## Installation Steps

1. Install Arduino IDE
2. Install ESP8266 Board Package
3. Connect NodeMCU via USB
4. Select Board: NodeMCU 1.0 (ESP-12E Module)
5. Upload the code



## Usage

1. Power the NodeMCU
2. Connect to WiFi network:

RAM_Server

Password:

12345678

3. Open browser and enter:

192.168.4.1

4. Login page will appear
